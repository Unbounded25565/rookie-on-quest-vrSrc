# 🛑 ROADMAP - PROJECT CEASED / END OF LIFE

**Rookie On Quest is officially discontinued as of March 19, 2026.**

Following the recent DMCA takedown by Meta (March 17, 2026) and the subsequent complete cessation of **VR Pirates (VRP)** and the **Rookie Sideloader** ecosystem, this project is no longer functional. All future development, including planned features and epics, has been **cancelled**.

---

## 🏁 Final Status (v3.0.0)

The project reached version **3.0.0** before its cessation. Key features implemented up to the final release included:

### ✅ Core Infrastructure (Completed)
- **Standalone Sideloading**: Native APK+OBB installation without a PC.
- **Persistent Queue System**: Reliable downloads with resume support.
- **Extraction Engine**: Progress-tracked 7z extraction using Apache Commons Compress.
- **Room Database**: Local caching of game data and installation history.

### ✅ User Experience (Completed)
- **Advanced Sorting & Search**: Filtering by name, size, and date.
- **Favorites System**: Quick access to preferred titles.
- **Unified Progress UI**: Clear feedback during all phases of the install process.

---

## 🛑 Cancelled / Abandoned Features

The following features were in progress or planned but will **not** be implemented:

- **Shizuku Integration**: Support for silent installations (Cancelled).
- **WorkManager Implementation**: Robust background persistence across app restarts (Cancelled).
- **Immersive VR Grid Mode**: Large poster-based UI for headset navigation.
- **Stickman Animation Machine**: Phase-specific contextual animations.
- **Automated Title Cleaning**: Regex-based removal of technical tags from game names.
- **In-App Notifications**: Real-time alerts for installation completion.

---

## 💡 Historical Note

> **Reason for Cessation**: This project was entirely dependent on the servers and infrastructure maintained by the **VRPirates/Rookie** team. On March 17, 2026, those servers were permanently taken offline following legal action from Meta. As a result, Rookie On Quest is unable to function and development has been terminated.
